## 백업 슬라이드 (질문 대비)

- **B1. ASIC 해석의 반전 과정** — 20000ps "동률" → 15000ps "v2 5배" → 10000ps 수렴:
  "왜 중간에 v2가 좋아 보였나" 질문 방어 (합성기가 덜 조여진 상태의 잔상, sweep_report §4)
- **B2. Fmax 방어** — "RTL 고쳤는데 116/146 유효?" → 골든 재빌드 WNS·자원 동일,
  크리티컬 패스 불변 (sweep_summary 재빌드 검증 절)
- **B3. Nitro 중단 근거** — SDA101 내부 assertion 2회 재현, 시도 기록 (log 47 §4)
- **B4. 전력 측정 방식 선정** — rail 직접 측정 배제(비파괴 원칙) → 5V 입력 차분 (log 46)
- **B5. scenario 1-1/1-2 상세 표** — 톤별 감쇠·판정 전체
