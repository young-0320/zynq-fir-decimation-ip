# FIR N43 — 보드 전력 실측 vs Vivado 추정 (Track B, §8-3)

- 측정일: 2026-07-22, 장비: OWON HDS242 (DMM 모드, 10A 잭 직렬)
- 대상: v2@145MHz BOOT.bin (`build/fir_n43_v2_freq_145mhz/output/`), 시나리오 `mode 1-1`
- 경로: 보드 5V 배럴잭 입력 (5V/2A 어댑터 + DC잭 피그테일 절단점, 전원 점퍼 WALL,
  micro-USB는 UART 전용) — 방식 의사결정·원 판독값은 `docs/log/46` §1~§5
- 성격: **보드 전체 입력 전력의 실측 보조 지표.** Vivado on-chip 추정과 측정 범위가
  다르므로(아래 §3) 절대 일치가 아니라 차이 요인의 분해·정합 확인이 목적.

## 1. 측정 결과

상태당 전류 5회 판독 중앙값, 전압은 부하 상태에서 상태별 실측. 전 상태 USB-UART 연결.

| 상태 | 조건 | I_med [A] | V [V] | P [W] |
| ---- | ---- | --------: | ----: | ----: |
| S0 | SD 미삽입 — BootROM 대기, PL 미구성 (baseline) | 0.340 | 5.047 | **1.72** |
| S1 | SD boot 완료 — PL 구성 + 145MHz 클럭, FIR idle | 0.443 | 4.988 | **2.21** |
| S2 | `mode 1-1` one-shot 반복 실행 중 | 0.435 | 5.008 | **2.18** |

## 2. 차분 해석

| 차분 | 값 | 의미 |
| ---- | --: | ---- |
| S1 − S0 | **+0.49 W** | PL bitstream 구성 + 145MHz 클럭(MMCM) + bare-metal 앱 + DDR 활성의 합 |
| S2 − S1 | **−0.03 W ≈ 0** | FIR+DMA burst(83µs)는 DMM 평균에 미검출 — 사전 예측(log 46 §1) 그대로 |

- S1−S0는 **PL 단독이 아니라** 위 네 요소의 합이다 (측정 방식 채택 시점에 인지된 한계,
  log 46 §1).
- S2 ≈ S1은 실패가 아니라 결과다: 8192샘플 처리가 83µs에 끝나 1~2초 주기의 데모
  사이클에서 평균 전력에 기여하는 몫이 측정 분해능 아래라는 것 — 연산 경로의 효율을
  전력 측면에서도 보여준다.

## 3. Vivado 추정 대비 분해

비교 기준: **1.705W** (v2@145MHz clk_wiz/MMCM 빌드, on-chip total —
`vivado/reports/sweep_summary_v2.md`).

| 단계 | 값 | 근거 |
| ---- | --: | ---- |
| 보드 실측 S1 (USB 포함) | 2.21 W | §1 |
| − USB-UART(FTDI) 몫 | −0.22 W | **실측 분리** (§4) |
| = 보드 실측 S1 (USB 제외 환산) | 1.99 W | USB 미연결 S1 0.398A 실측과 일치 |
| ÷ 레귤레이터 효율 85~90% (가정) | 1.69~1.79 W | 5V→코어/IO 레일 변환 손실 |
| Vivado on-chip 추정 | **1.705 W** | 위 구간 안 → **정합 범위** |

**결론**: 보드 실측(2.21W)이 추정(1.705W)보다 +0.5W 큰 것은 방향·크기 모두 설명된다
— USB-UART 0.22W는 실측으로 직접 분리되었고, 잔여 차이는 레귤레이터 변환 손실 및
추정 범위 밖 주변부(DDR3 칩, 이더넷 PHY 등) 몫으로 해석된다.

한계 (정직 고지):
- 레귤레이터 효율 85~90%는 가정값이다 (레일별 부하 배분을 모르면 정밀화 불가).
- DDR3 칩 등 주변부 소비는 Vivado 추정에 없고 실측에는 포함되므로, "USB 제외 환산
  1.99W × 효율 ≈ 추정 1.705W"는 등식이 아니라 **정합성 점검**이다. 주변부와 효율
  손실이 서로 상쇄되는 방향으로 섞여 있어 개별 분리는 이 측정 범위 밖.
- S0의 PS 상태(BootROM 대기)는 Vivado 추정의 PS 가정(앱 구동)과 다르므로 S1−S0
  0.49W를 Vivado의 PL·MMCM 증분과 직접 등치할 수 없다 — 규모(수백 mW)의 정합만 확인.

## 4. 부가 관찰 — USB-UART 관측 인프라의 전력 비용

| 상태 | USB 미연결 | USB 연결 | 오프셋 |
| ---- | --------: | -------: | -----: |
| S0 | 0.293 A | 0.340 A | +47 mA |
| S1 | 0.398 A | 0.443 A | +45 mA |

- USB-UART(FTDI 브리지) 연결 비용 ≈ **0.22W** — FIR 연산이 평균 전력에 남기는 몫(미검출)
  보다 크다. "UART는 처리 경로가 아니라 관측 경로"라는 설계 구분이 전력 데이터로도
  확인된다.
- USB 유무 양 조건에서 S1−S0 델타가 재현(0.105 vs 0.103A)되어 **차분 프로토콜의
  유효성**을 스스로 검증한다.

기타 관찰: 어댑터 무부하 5.183V → 부하 시 4.99~5.05V(정상 강하), 전원 인가 직후
~0.30A 과도 후 안정화(판독은 안정화 후), S2 중 순간 0.456A(UART 전송 활동 추정,
중앙값이 흡수). `mode 1-1` reset 없이 반복 실행 정상 — 반복 실행 경로 보드 확인 겸.

## 5. 요약 (보고서·발표 인용용)

> 보드 5V 입력 실측: S0 1.72W / S1 2.21W / S2 2.18W. 차분 S1−S0 = 0.49W(PL 구성+클럭
> +앱+DDR), S2−S1 ≈ 0(µs burst 미검출 — 예측 부합). USB-UART 0.22W를 실측 분리하고
> 레귤레이터 효율(85~90%)을 감안하면 Vivado on-chip 추정 1.705W와 정합 범위.

## 부록 B — 측정 결선 구성

### 공통 토폴로지

DC잭 암·수 피그테일 쌍으로 +5V 급전 경로에 개폐 가능한 **절단점 M**을 확보한다.
보드·케이블 무개조, GND는 전 과정 직결 유지.

```
5V 어댑터 ──[DC잭 암]── +5V ──〈 M: 절단점 〉── +5V ──[DC잭 수]── 보드 배럴잭
                        GND ────── 상시 직결 ────── GND        (전원 점퍼: WALL)
```

### (1) 전류 측정 — 절단점 M에 DMM 직렬 삽입 (DC A 모드)

전류계는 부하 전류가 계기 내부 션트를 **관통**해야 하므로 +5V 경로에 직렬로 넣는다.

```
··· +5V ──▶ 적색 리드 [10A 잭] ── HDS242 (DC A) ── [COM] 흑색 리드 ──▶ +5V ···
    GND ─────────────────────────── 직결 ─────────────────────────── GND
```

- **10A 전용 잭** 사용 — 예상 전류 0.4~0.7A는 mA 잭 퓨즈 정격을 초과.
- 두 리드 모두 +5V 경로 위의 접점이다(입구/출구). 리드 방향은 표시 부호만 결정.
- 접점은 리드 침에 도선을 권선 후 절연 — 접촉 이탈은 곧 보드 전원 차단이므로 고정 필수.

### (2) 전압 측정 — M 직결 복원 후 병렬 접촉 (DC V 모드)

DMM을 경로에서 제거하고 M을 직결(브리지)로 복원하면 보드는 계기 없이 급전된다.
전압계는 고임피던스 **병렬** 측정이므로 두 노드에 순간 접촉으로 충분하다.

```
+5V 경로: ··· ──●── ···   ●: 브리지 매듭  ◀── 적색 리드 [VΩ 잭]
GND 경로: ··· ──●── ···   ●: 직결 매듭    ◀── 흑색 리드 [COM]
                          HDS242 (DC V), 부하 상태에서 상태별 측정
```

- 부하 중 보드 입력단 전압을 재므로 어댑터 무부하값(5.183V)이 아닌 실효 공급
  전압(4.99~5.05V)이 얻어진다 — P=V×I의 V로 이 값을 사용.

### 전환 규칙 (단일 계기 운용의 핵심)

| 단계 | 적색 리드 잭 | 모드 | M 상태 |
| --- | --- | --- | --- |
| 전류 측정 | **10A** | DC A | DMM 직렬 |
| 전압 측정 | **VΩ** | DC V | 직결 브리지 |

- 전류→전압 전환 시 적색 리드를 **반드시 10A → VΩ 잭으로 이동**. 10A 잭은 저임피던스
  션트라 그 상태로 전압원에 병렬 접촉하면 단락이다.
- 모든 결선 변경은 어댑터 분리 후 수행.
