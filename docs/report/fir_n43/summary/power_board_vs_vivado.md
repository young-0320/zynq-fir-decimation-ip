# FIR N43 — 보드 전력 실측 vs Vivado 추정 (Track B, §8-3)

- 측정일: 2026-07-22, 장비: OWON HDS242 (DMM 모드, 10A 잭 직렬)
- 대상: v2@145MHz BOOT.bin (`build/fir_n43_v2_freq_145mhz/output/`), 시나리오 `mode 1-1`
- 경로: 보드 5V 배럴잭 입력 (5V/2A 어댑터 + DC잭 피그테일 절단점, 전원 점퍼 WALL,
  micro-USB는 UART 전용) — 방식 의사결정·측정 당일 기록은 `docs/log/46`
- 성격: 보드 전체 입력 전력의 실측 보조 지표. Vivado on-chip 추정과는 측정 범위가
  다르므로(§3) 절대치 일치가 아니라 차이 요인의 분해와 정합성 확인이 목적이다.

## 1. 측정 결과

상태당 전류 5회 판독의 중앙값을 취하고, 전압은 부하 상태에서 상태별로 측정했다.
세 상태 모두 USB-UART(PC) 연결 기준이다.

| 상태 | 조건 | I_med [A] | V [V] | P [W] |
| ---- | ---- | --------: | ----: | ----: |
| S0 | SD 미삽입 — BootROM 대기, PL 미구성 (baseline) | 0.340 | 5.047 | 1.72 |
| S1 | SD boot 완료 — PL 구성 + 145MHz 클럭, FIR idle | 0.443 | 4.988 | 2.21 |
| S2 | `mode 1-1` one-shot 반복 실행 중 | 0.435 | 5.008 | 2.18 |

## 2. 차분 해석

| 차분 | 값 | 의미 |
| ---- | --: | ---- |
| S1 − S0 | +0.49 W | PL bitstream 구성 + 145MHz 클럭(MMCM) + bare-metal 앱 + DDR 활성의 합 |
| S2 − S1 | −0.03 W (≈ 0) | FIR+DMA burst(83µs)는 DMM 평균에 나타나지 않음 |

- S1−S0는 PL 단독이 아니라 위 네 요소의 합이다. 이는 측정 방식을 정할 때부터 인지된
  한계다(log 46 §1).
- S2−S1이 0에 가까운 것은 사전 예상과 일치하는 결과다. 8192샘플 처리가 83µs에 끝나므로
  1~2초 주기의 데모 사이클에서 평균 전력에 기여하는 몫이 측정 분해능 아래에 있다.
  연산 경로의 처리 효율을 전력 측면에서 보여주는 수치이기도 하다.

## 3. Vivado 추정 대비 분해

비교 기준: 1.705W (v2@145MHz clk_wiz/MMCM 빌드의 on-chip total,
`vivado/reports/sweep_summary_v2.md`).

| 단계 | 값 | 근거 |
| ---- | --: | ---- |
| 보드 실측 S1 (USB 포함) | 2.21 W | §1 |
| − USB-UART(FTDI) 몫 | −0.22 W | 실측으로 분리 (§4) |
| = 보드 실측 S1 (USB 제외 환산) | 1.99 W | USB 미연결 S1 0.398A 실측과 일치 |
| ÷ 레귤레이터 효율 85~90% (가정) | 1.69~1.79 W | 5V→코어/IO 레일 변환 손실 |
| Vivado on-chip 추정 | 1.705 W | 위 구간 안 — 정합 범위 |

보드 실측(2.21W)이 추정(1.705W)보다 0.5W 큰 것은 방향과 크기가 모두 설명된다.
USB-UART 0.22W는 실측으로 직접 분리했고, 잔여 차이는 레귤레이터 변환 손실과
추정 범위 밖 주변부(DDR3 칩, 이더넷 PHY 등)의 몫으로 해석된다.

한계:

- 레귤레이터 효율 85~90%는 가정값이다. 레일별 부하 배분을 알 수 없어 더 정밀하게
  잡을 수 없다.
- DDR3 칩 등 주변부 소비는 Vivado 추정에 없고 실측에는 포함된다. 따라서 위 분해는
  등식이 아니라 정합성 점검이며, 주변부와 효율 손실의 개별 분리는 이 측정 범위 밖이다.
- S0의 PS 상태(BootROM 대기)는 Vivado 추정의 PS 가정과 다르다. S1−S0 0.49W를 Vivado의
  PL·MMCM 증분과 직접 등치할 수는 없고, 수백 mW라는 규모의 정합만 확인한다.

## 4. 부가 관찰 — USB-UART 인터페이스의 전력

| 상태 | USB 미연결 | USB 연결 | 오프셋 |
| ---- | --------: | -------: | -----: |
| S0 | 0.293 A | 0.340 A | +47 mA |
| S1 | 0.398 A | 0.443 A | +45 mA |

- USB-UART(FTDI 브리지)를 PC에 연결하면 약 0.22W가 추가된다. FIR 연산이 평균 전력에
  남기는 몫(미검출)보다 크다. 즉 처리 경로가 아닌 관측 경로(UART)가 연산보다 전력을
  더 쓴다 — "UART는 관측 경로"라는 설계 구분과 부합하는 결과다.
- USB 유무 두 조건에서 S1−S0 델타가 각각 0.105A / 0.103A로 재현되어, 차분 측정의
  재현성을 확인했다.

기타 관찰: 어댑터 무부하 전압 5.183V는 부하 시 4.99~5.05V로 낮아진다(정상적인 부하
강하). 전원 인가 직후 약 0.30A의 과도 전류가 흐른 뒤 안정화되며, 판독은 안정화 후에
수행했다. S2 중 순간적으로 0.456A가 관측되었는데 UART 전송 구간의 활동으로 추정되며,
중앙값 채택으로 이상치의 영향을 배제했다. `mode 1-1`은 reset 없이 반복 실행에서
전 회 정상 완료되어, 반복 실행 경로의 보드 확인을 겸했다.

## 5. 요약

> 보드 5V 입력 실측: S0 1.72W / S1 2.21W / S2 2.18W. 차분 S1−S0 = 0.49W(PL 구성+클럭
> +앱+DDR), S2−S1 ≈ 0(µs burst 미검출 — 사전 예상과 일치). USB-UART 0.22W를 실측
> 분리하고 레귤레이터 효율(85~90%)을 감안하면 Vivado on-chip 추정 1.705W와 정합 범위.

## 부록 B — 측정 결선 구성

### 공통 토폴로지

DC잭 암·수 피그테일 쌍으로 +5V 급전 경로에 개폐 가능한 절단점 M을 확보한다.
보드·케이블은 개조하지 않으며, GND는 전 과정에서 직결을 유지한다.

```
5V 어댑터 ──[DC잭 암]── +5V ──〈 M: 절단점 〉── +5V ──[DC잭 수]── 보드 배럴잭
                        GND ────── 상시 직결 ────── GND        (전원 점퍼: WALL)
```

### (1) 전류 측정 — 절단점 M에 DMM 직렬 삽입 (DC A 모드)

전류계는 부하 전류가 계기 내부 션트를 관통해야 하므로 +5V 경로에 직렬로 넣는다.

```
··· +5V ──▶ 적색 리드 [10A 잭] ── HDS242 (DC A) ── [COM] 흑색 리드 ──▶ +5V ···
    GND ─────────────────────────── 직결 ─────────────────────────── GND
```

- 10A 전용 잭을 사용한다. 예상 전류 0.4~0.7A는 mA 잭의 퓨즈 정격을 초과한다.
- 두 리드 모두 +5V 경로 위의 접점이다(입구/출구). 리드 방향은 표시 부호만 결정한다.
- 접점은 리드 침에 도선을 감아 절연 테이프로 고정한다. 접촉 이탈은 곧 보드 전원
  차단이므로 고정 상태를 확인하고 부팅한다.

### (2) 전압 측정 — M 직결 복원 후 병렬 접촉 (DC V 모드)

DMM을 경로에서 제거하고 M을 직결(브리지)로 복원하면 보드는 계기 없이 급전된다.
전압계는 고임피던스 병렬 측정이므로 두 노드에 순간 접촉으로 충분하다.

```
+5V 경로: ··· ──●── ···   ●: 브리지 매듭  ◀── 적색 리드 [VΩ 잭]
GND 경로: ··· ──●── ···   ●: 직결 매듭    ◀── 흑색 리드 [COM]
                          HDS242 (DC V), 부하 상태에서 상태별 측정
```

- 부하 중 보드 입력단 전압을 재므로 어댑터 무부하값(5.183V)이 아닌 실효 공급
  전압(4.99~5.05V)이 얻어진다. P=V×I의 V로 이 값을 사용한다.

### 전환 규칙

| 단계 | 적색 리드 잭 | 모드 | M 상태 |
| --- | --- | --- | --- |
| 전류 측정 | 10A | DC A | DMM 직렬 |
| 전압 측정 | VΩ | DC V | 직결 브리지 |

- 전류에서 전압으로 전환할 때 적색 리드를 10A에서 VΩ 잭으로 옮겨야 한다. 10A 잭은
  저임피던스 션트라서 그 상태로 전압원에 병렬 접촉하면 단락이 된다.
- 모든 결선 변경은 어댑터를 분리한 뒤 수행한다.
