# FIR Decimation Input Signal Spec

## 1. 목적

본 문서는 ideal/fixed/RTL 비교에 공통으로 사용할 입력 신호 생성 제약을 정의한다.

- 대상: `Fs_in = 100 MHz` 환경에서의 멀티톤 입력 자극
- 용도: alias 관찰, ideal 검증, fixed Q-format 결정, RTL 비교의 공통 기준선
- 본 문서에서 입력 신호 제약이 확정된 뒤에 fixed Q-format을 최종 결정한다

## 2. 시스템 맥락

- 입력 샘플링 주파수: `Fs_in = 100e6` (Hz)
- 디시메이션 계수: `M = 2`
- 출력 샘플링 주파수: `Fs_out = 50e6` (Hz)
- 통과대역 경계: `fp = 15e6` (Hz)
- 저지대역 시작: `fs = 25e6` (Hz)
- 목표 감쇠: `As >= 60 dB`

## 5. fixed Q-format과의 관계

- 아래 항목이 확정된 뒤에만 Q-format을 최종 확정한다:
  - tone 개수
  - 진폭 배분
  - peak/RMS 목표
  - headroom
  - phase 정책
- 그전까지의 Q-format 후보는 작업 가정으로만 취급한다.

## 6. 검증 기준

- 동일 파라미터로 동일 입력이 재생성 가능해야 한다.
- alias 관찰 목적의 톤 배치가 문서에 설명되어 있어야 한다.
- ideal 검증과 fixed 동적 범위 분석에 필요한 신호 통계가 기록되어 있어야 한다.
- clipping 여부와 peak/RMS 값이 문서에 남아 있어야 한다.
