# docs/workflow — 단계별 실행 계획

이 폴더는 각 시점의 목표·준비물·실행 절차를 담은 계획 문서다. v1부터 v24까지 번호가
곧 프로젝트의 진행 순서이며, 새 국면마다 새 버전을 만들고 이전 버전은 수정하지 않고
보존한다. 결정의 근거와 결과는 계획이 아니라 `../log/`에 기록한다 — 계획(workflow)과
기록(log)의 역할 분리가 이 레포의 문서 규칙이다.

기록 원본과 마찬가지로 AI 협업 워크플로우로 작성되었고 사후 가공 없이 보존한다.
완료 표시(✅)와 갱신 주석이 남아 있는 것은 당시 상태를 그대로 남긴 것이다.

- 최신: `workflow_v24.md` — 마무리·공개 체크리스트 (마지막 워크플로우)
- 항목별 최종 상태는 `workflow_v23.md` 하단 상태 표 참고
